1. **View PRs**: To view pull requests, enter:
   ```bash
   gh pr list
   ```

2. **Review a PR**: To review a specific pull request, use:
   ```bash
   gh pr review [PR_NUMBER]
   ```

3. **Comment on a Review**: To add a comment, run:
   ```bash

   ```

4. **Resolve Comments**: To resolve a comment, use:
   ```bash
   gh pr resolve [COMMENT_ID]
   ```

These commands help streamline your review process and allow for quick navigation through pull request threads.
